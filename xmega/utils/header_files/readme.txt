Installation instructions for patched XMEGA device header files.


GCC (WinAVR):
-------------
The target directory is the \avr\include\avr\ subdirectory of your WinAVR
install directory, e.g., C:\WinAVR-20100110\avr\include\avr\.

Optional step: back up either the entire directory or just the iox*.h files.

Copy the contents of the \gcc\ subdirectory to the target directory. Windows
will ask for confirmation whether you want to overwrite the existing files or
not. Select Yes to all.


IAR EWAVR:
-------------
The target directory is the \avr\inc\Atmel\ subdirectory of your IAR EWAVR
install directory, e.g., C:\Program Files\IAR Systems\Embedded Workbench 5.x\
avr\inc\Atmel\.

Optional step: back up the entire directory.

Copy the contents of the \iar\ subdirectory to the target directory. Windows
will ask for confirmation whether you want to overwrite the existing files or
not. Select Yes to all.
